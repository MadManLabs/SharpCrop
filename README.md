# SharpCrop

This application can grab a part of your screen and upload it to your Dropbox / Google Drive / OneDrive cloud.

## Usage

* `Esc` Close the crop window
* `F1` Open config window
* `Left Click` Image capture
* `Right Click` Gif capture
