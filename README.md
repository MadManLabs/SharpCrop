# SharpCrop

This application can capture a part of your screen and upload it to the cloud - or save it locally.

## Providers

* Dropbox
* Google Drive
* OneDrive
* FTP
* Local FS

## Languages

* English
* Hungarian

## Usage

* `Esc` Close the crop window
* `F1` Open config window
* `Left` Image capture
* `Right` Video capture
